AI Interior Design Web Application
📌 Project Overview

The AI Interior Design Web Application is a smart web-based project that helps users generate interior design ideas based on selected styles such as Modern, Luxury, and Minimal. It provides an interactive experience where users can click a style and view AI-inspired room designs instantly.

🚀 Features
🎨 Generate interior designs based on user selection
🏠 Style categories: Modern, Luxury, Minimal
⚡ Instant UI-based design preview
💡 Clean and responsive user interface
🖱️ Interactive button-based design generation
🌐 Built using only frontend technologies
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
AI Logic (Simulation): JavaScript-based design switching
Styling: CSS Flexbox / Grid / Animations
Optional Integration: AI APIs (future enhancement)

⚙️ How to Run the Project
# Step 1: Download or clone the repository
git clone https://github.com/trisha-iragattapu/ai-interior-design.git

# Step 2: Open project folder
cd ai-interior-design

# Step 3: Open index.html in browser

👉 Simply double-click index.html or use Live Server in VS Code.

🎯 How It Works
User opens the website
Selects a style (Modern / Luxury / Minimal)
JavaScript detects the selection
Corresponding interior design is displayed
UI updates dynamically without page reload
🖼️ Sample Features
Modern Interior → Clean, simple, aesthetic rooms
Luxury Interior → Premium furniture and elegant designs
Minimal Interior → Simple and space-efficient layouts
